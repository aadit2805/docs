Sift is a monorepo with two packages — `client/` (Next.js) and `server/` (Express) — run concurrently from the project root.

<Info>
  **Prerequisites**:

  - Node.js 19\+
  - `make` (preinstalled on macOS/Linux; on Windows use WSL)
  - Supabase, Clerk, and Anthropic API credentials
</Info>

## Install

```bash
make install
```

Installs dependencies for both `client/` and `server/`.

## Run the dev servers

```bash
make dev
```

- Frontend on `http://localhost:3000`
- Backend on `http://localhost:3001`

<Note>
  `make dev` runs `tsx` one-shot — the server does **not** auto-reload. For server auto-reload during development, use `npm run dev` from the project root instead. It uses `concurrently` and the server's `tsx watch`.
</Note>

## Common tasks

| Command | Purpose |
| --- | --- |
| `make client` | Frontend only |
| `make server` | Backend only |
| `make scrape` | Run the data scraper (Anex \+ RateMyProfessors → Supabase) |
| `make clean` | Remove build artifacts and `node_modules` |

Build \+ type check:

```bash
cd client && npx next build       # Frontend build + type check
cd server && npx tsc --noEmit     # Backend type check
cd client && npm run lint         # Frontend lint
```

There are no tests in this codebase — no test framework is configured.

## Environment variables

### `client/.env.local`

```bash
NEXT_PUBLIC_API_URL=http://localhost:3001
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=...
CLERK_SECRET_KEY=...
NEXT_PUBLIC_CLERK_SIGN_IN_URL=/sign-in
NEXT_PUBLIC_CLERK_SIGN_UP_URL=/sign-up
```

### `server/.env`

```bash
PORT=3001
FRONTEND_URL=http://localhost:3000
SUPABASE_URL=...
SUPABASE_ANON_KEY=...
ANTHROPIC_API_KEY=...
CLERK_SECRET_KEY=...
CLERK_PUBLISHABLE_KEY=...
```

## Hosting

- **Frontend** — Vercel (Next.js)
- **Backend** — Railway / Fly.io (Express). Note: backend trusts the proxy so rate limits work behind Railway.
