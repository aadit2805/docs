---
title: 'List users'
description: 'Retrieve a paginated list of users.'
---

## Request

```http
GET /users
```

### Query parameters

<ParamField query="limit" type="integer" default="20">
  Maximum number of users to return. Capped at `100`.
</ParamField>

<ParamField query="offset" type="integer" default="0">
  Number of users to skip before starting to return results. Use with `limit` to paginate.
</ParamField>

## Response

<ResponseField name="limit" type="integer">
  The `limit` applied to the response.
</ResponseField>

<ResponseField name="offset" type="integer">
  The `offset` applied to the response.
</ResponseField>

<ResponseField name="users" type="array">
  The list of users for the current page.

  <Expandable title="User">
    <ResponseField name="id" type="integer">
      Unique identifier for the user.
    </ResponseField>
    <ResponseField name="name" type="string">
      The user's name.
    </ResponseField>
  </Expandable>
</ResponseField>

### Example

```bash cURL
curl "https://api.example.com/users?limit=2&offset=0"
```

```json Response
{
  "limit": 2,
  "offset": 0,
  "users": [
    { "id": 1, "name": "user_1" },
    { "id": 2, "name": "user_2" }
  ]
}
```
