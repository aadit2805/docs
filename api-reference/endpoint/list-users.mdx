---
title: 'List users'
description: 'Retrieve a paginated list of users.'
---

Returns users in insertion order. Use `limit` and `offset` to page through results.

## Request

```http
GET /users
```

### Query parameters

<ParamField query="limit" type="integer" default="20">
  Maximum number of users to return in a single response. Must be between `1` and `100`. Values above `100` are clamped to `100`.
</ParamField>

<ParamField query="offset" type="integer" default="0">
  Number of users to skip before starting to return results. Combine with `limit` to paginate through the full collection.
</ParamField>

## Response

<ResponseField name="limit" type="integer">
  The effective `limit` applied to this response after clamping.
</ResponseField>

<ResponseField name="offset" type="integer">
  The `offset` applied to this response.
</ResponseField>

<ResponseField name="users" type="array">
  The list of users for the current page. May be empty if `offset` is beyond the end of the collection.

  <Expandable title="User">
    <ResponseField name="id" type="integer">
      Unique identifier for the user.
    </ResponseField>
    <ResponseField name="name" type="string">
      The user's name.
    </ResponseField>
  </Expandable>
</ResponseField>

## Pagination

To walk the full collection, start at `offset=0` and increment `offset` by `limit` until the returned `users` array is shorter than `limit` (or empty).

<CodeGroup>

```bash cURL
curl "https://api.example.com/users?limit=50&offset=0"
curl "https://api.example.com/users?limit=50&offset=50"
```

```javascript Node.js
let offset = 0;
const limit = 50;

while (true) {
  const res = await fetch(`https://api.example.com/users?limit=${limit}&offset=${offset}`);
  const { users } = await res.json();

  for (const user of users) {
    console.log(user);
  }

  if (users.length < limit) break;
  offset += limit;
}
```

```python Python
import requests

offset, limit = 0, 50

while True:
    res = requests.get(
        "https://api.example.com/users",
        params={"limit": limit, "offset": offset},
    ).json()

    for user in res["users"]:
        print(user)

    if len(res["users"]) < limit:
        break
    offset += limit
```

</CodeGroup>

## Example

<CodeGroup>

```bash cURL
curl "https://api.example.com/users?limit=2&offset=0"
```

```javascript Node.js
const res = await fetch("https://api.example.com/users?limit=2&offset=0");
const data = await res.json();
```

```python Python
import requests

data = requests.get(
    "https://api.example.com/users",
    params={"limit": 2, "offset": 0},
).json()
```
